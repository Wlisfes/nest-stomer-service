# nest-plumb-service

## 基于 nestjs 构建的流程图服务端

> [在线演示地址](http://demo.lisfes.cn)

## 如何安装与使用

```
git clone https://github.com/Wlisfes/nest-plumb-service.git

cd nest-plumb-service

npm install

npm run dev

```

### 根据.env.example 案例在根吗目录添加配置文件.env

```
#mysql
ORM_TYPE =
ORM_HOST =
ORM_PORT =
ORM_USERNAME =
ORM_PASSWORD =
ORM_DATABASE =
ORM_CHARSET =
ORM_SYNCHRONIZE =
ORM_DATESTRINGS =
```
